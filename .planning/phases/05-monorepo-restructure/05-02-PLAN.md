**packages/relay/src/index.ts:**
Create the relay service entry point based on the existing scripts/push-stream.ts. The relay service will:

1. Poll `/api/stream/demand` every ~10s using setTimeout chaining (per STATE.md decisions)
2. Start ffmpeg (RTSP→RTMPS) when `shouldStream` is true
3. Stop ffmpeg (SIGTERM with SIGKILL fallback) when demand expires
4. Track consecutive failures; stop after ~5min of failures (safety)
5. Write relay status to KV via Worker endpoint
